# Acceptance

## User Steps

1. Open the sidebar Diva pet full-screen module with no prior scene preference.
2. Confirm the scene starts as transparent background.
3. Select a non-transparent scene such as Sea or Space.
4. Restart or reload the GUI.
5. Confirm the selected scene remains active and was not reset by backend config loading.

## Expected Result

- Scene selection persists as frontend GUI configuration.
- The core backend config does not gain Gauss scene selection fields.
