# 10 - VRM UI 设计文档

> VRM 3D 角色的界面设计与交互规范

---

## 1. 设计理念

VRM 角色采用全 3D 渲染，支持用户自由旋转视角、缩放角色。设计原则：

1. **沉浸式 3D**：用户可通过鼠标拖拽旋转视角，滚轮缩放
2. **表情可视化**：角色表情随对话内容实时变化
3. **口型同步**：TTS 播报时嘴巴同步开合
4. **简洁控制**：控制面板不遮挡角色

## 3. 交互设计

### 3.1 鼠标交互

| 操作 | 行为 |
|------|------|
| 左键拖拽 | 旋转视角 (OrbitControls) |
| 滚轮 | 缩放 (0.5x - 3.0x) |
| 右键拖拽 | 平移视角 |
| 双击角色 | 重置视角 |

### 3.2 表情切换

底部控制栏提供快捷表情按钮：

```
[😊 开心] [😢 悲伤] [😲 惊讶] [😠 生气] [😐 中性]
```

手动点击可 override 自动情绪推断。

### 3.3 动作菜单

```
[🎬 动作 ▼]
├── 打招呼
├── 比耶
├── 伸展
├── 挠头
└── 自定义...
```

## 4. 状态视觉

| 状态 | VRM 表情 | 动画 |
|------|---------|------|
| 空闲 | neutral + 1.0 | idle 动画 |
| 等待回复 | neutral + 0.5, blink | idle |
| Agent 思考 | surprised + 0.3 | scratch_head (挠头) |
| 开心回复 | happy + 1.0 | greeting |
| 难过回复 | sad + 1.0 | — |
| TTS 播报中 | mouth sync | talking idle |
| 用户拖拽 | surprised + 0.5 | — |
| 错误 | sad + 0.5 | — |

## 6. 模型元数据显示

从 VRM 文件的 meta 信息中提取：

```typescript
const meta = vrm.meta
// meta.name         → "Alice"
// meta.author       → "VRoid Studio"
// meta.licenseName  → "CC0"
// meta.otherLicenseUrl → "https://..."
```

在设置面板底部显示许可信息，提示用户遵守模型许可。
