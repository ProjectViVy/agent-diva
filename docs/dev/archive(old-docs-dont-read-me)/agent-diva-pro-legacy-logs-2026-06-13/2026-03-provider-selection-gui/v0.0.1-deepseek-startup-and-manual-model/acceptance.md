# Acceptance

1. Start the GUI with default DeepSeek configuration and no pre-existing saved model shortcuts.
2. Confirm the titlebar model switcher shows the active DeepSeek model without first opening provider settings.
3. Send a message directly from the main page and confirm the active DeepSeek configuration is used without requiring manual reselection.
4. Open Settings -> Providers, select a provider, enter a model ID that is not listed in the catalog, and click "Add and Switch".
5. Confirm the new model becomes the current model immediately.
6. Return to the main page and confirm the newly added model appears in the titlebar switcher.
7. If two providers contain the same model name, confirm the titlebar checkmark/highlight tracks the active provider/model pair rather than model name alone.
