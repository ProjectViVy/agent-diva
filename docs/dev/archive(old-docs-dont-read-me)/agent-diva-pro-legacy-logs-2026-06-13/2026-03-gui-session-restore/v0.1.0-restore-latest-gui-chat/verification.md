## 建议的手动冒烟（Tauri + 已连接 Manager/运行时）

1. **双次冷启动**：在 GUI 中向当前聊天发送至少一条消息并等待落盘；完全退出应用后再次启动，应仍显示**同一会话**的历史消息，侧栏高亮/选中应与之一致。
2. **多频道干扰**：若存在非 `gui:` 会话且其 `updated_at` 比 GUI 会话更新，启动后仍应打开 **GUI 中最近更新** 的会话，而非其它频道会话。
3. **无 GUI 历史**：删除或清空 `~/.agent-diva/sessions/` 下全部 `gui_*.jsonl`（或等价干净环境）后启动，应仍为欢迎语 + 新会话，无报错阻塞。
