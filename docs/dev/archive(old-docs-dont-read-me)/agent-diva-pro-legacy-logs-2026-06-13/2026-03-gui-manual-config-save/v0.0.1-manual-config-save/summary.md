# Iteration Summary

## Scope

本次迭代把 `agent-diva-gui` 设置页中 `供应商`、`频道`、`网络` 三个配置区域从“字段变更即自动保存”调整为“本地草稿 + 手动保存按钮”模式。

## Delivered Changes

- 在供应商、频道、网络设置页右侧主区域头部新增统一的“保存配置”按钮。
- 三个设置页都改为维护本地草稿、最近一次成功保存快照、保存中状态与脏状态。
- 移除相关实时保存逻辑，不再因为输入、切换、开关变化而自动写后端。
- `App.vue` 中的配置保存入口改为可 `await` 的显式异步动作，并统一在成功时弹出绿色 toast“保存成功”。
- 新增轻量全局 toast 展示层与 store，成功保存时展示绿色提示，失败仍沿用现有错误弹窗。
- 新增频道单独保存入口，只保存当前选中的频道配置。

## Impact

- 配置编辑时不再频繁调用后端保存接口，交互更可控，也更适合连续调试与批量修改。
- 保存失败时不会丢失当前草稿，用户可以继续调整后再次保存。
