# Acceptance

1. 打开 GUI 设置页，进入“供应商”。
2. 点击左侧“新增供应商”按钮。
3. 输入供应商 ID、显示名称、API Base、默认模型，可选填写 API Key。
4. 点击“新增供应商”后，确认新供应商出现在左侧列表并被自动选中。
5. 确认该供应商的默认模型已成为当前模型，且已写入快捷模型列表。
6. 重新打开设置页后，确认该自定义供应商仍然存在。
7. 在任意供应商中点击“手动添加模型”，输入一个未预置的模型 ID。
8. 点击“添加并切换”后，确认该模型立即出现在模型列表中，并在重新打开设置页后仍然保留。
9. 对手动添加的模型，确认列表右侧出现删除按钮。
10. 点击删除按钮后，确认该模型从列表和快捷模型中移除，并在重新打开设置页后不再出现。
