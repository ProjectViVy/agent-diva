# Acceptance

Sprint 5 is accepted when:

- the failure validation matrix is recorded and linked from project management
- Mentle startup/open failure falls back without prompt/tool mismatch
- recall/search failure is non-fatal to the main dialogue flow
- diary/write failure keeps file-backed memory authoritative
- invalid tool definitions are skipped individually
- runtime active state follows actual `memtle_status` availability
- cron/default rebuilds preserve Mentle tools and prompt state
- `with_toolset()` remains registry-driven
- subagents remain Mentle-disabled by default
- default-lane and Mentle feature checks are both available locally and in CI
- CI continues to reject local path/git/patch overrides for `memtle`

Sprint 5 does not require Sprint 7 GUI or tool-selection controls.
