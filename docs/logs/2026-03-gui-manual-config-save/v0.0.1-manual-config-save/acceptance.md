# Acceptance

## Expected User Acceptance Steps

1. 打开桌面端设置页，进入 `供应商`、`频道`、`网络` 三个分区。
2. 修改任意字段，确认页面右上出现可点击的“保存配置”按钮，且此时不应立即触发后端保存。
3. 点击“保存配置”：
   - 成功时显示绿色 toast“保存成功”；
   - 失败时显示错误弹窗，且当前草稿仍保留。
4. 在频道页切换不同频道后再切回，确认未保存草稿仍保留在页面内存中，且切换过程不会隐式保存。
5. 在网络页修改 `provider`、`api_key`、`max_results` 后，确认只有点击保存才会写回后端。

## Acceptance Notes

- 用户主动触发的其他刷新能力（例如显式刷新模型列表）不在本轮移除范围内。
