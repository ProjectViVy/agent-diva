# Release: AutoDream Rhythm Distillation Design

> Date: 2026-05-30
> Version: v0.0.1-rhythm-distillation-design

## Release Method

This is a design specification document. No code release is required.

The document is committed to the repository under `docs/dev/genericagent/`. It is available for implementation teams to consume directly.

## Deployment Impact

- No runtime changes.
- No configuration changes.
- No dependency changes.
- No migration required.

## Follow-up

Implementation should follow the phased plan in Section 15 of the design document:

- P0: Manual trigger + session-end check + structured output.
- P1: Rhythm gates + subagent runner + candidate/journal integration.
- P2: Richer rhythm + Mentle + auto-merge + GUI.
