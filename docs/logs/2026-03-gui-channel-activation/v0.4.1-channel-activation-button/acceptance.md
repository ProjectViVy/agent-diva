# Acceptance

1. Open GUI settings and navigate to `频道`.
2. Confirm each channel row shows an action button to the right of the channel name.
3. Confirm disabled channels show `激活` and enabled channels show `取消激活`.
4. Confirm clicking the row selects the channel without changing its enabled state.
5. Confirm clicking the action button only changes the enabled state.
6. Confirm the detail header status pill reflects the new state and is display-only.
