# Acceptance

## User Steps

1. Open the Diva pet settings panel.
2. Confirm the tabs are 外观, VRM 模型, 动画.
3. Open 外观 and confirm 默认角色 appears as the selected default when no user appearance exists.
4. Confirm 默认角色 has no edit or delete action.
5. Open VRM 模型 and confirm `.vrm` import is available there.
6. Import or load a VRM model, then create a user appearance from 外观 using that model.
7. Delete the active user appearance and confirm the pet falls back to 默认角色 / Alice.
8. Open the desktop pet right-click menu and confirm appearance switching includes 默认角色.
