# 验收步骤

1. 打开 Diva 桌宠页，点击右上角设置按钮，确认标题为“外观设置”，页签为“外观 / 动画”。
2. 在“外观”页查看当前模型、内置 Alice 模型和自定义模型列表。
3. 导入 `.vrm` 文件，确认文件写入 `~/.agent-diva/vrm/models/custom/`，并可立即切换当前角色。
4. 新建外观，选择模型、待机动作、动画开关和表情开关，应用后确认配置同步生效。
5. 在“动画”页开关待机动画，选择待机动作集合。
6. 点击 one-shot 动作预览按钮，确认角色播放动作；再次点击可停止。
7. 桌面宠右键菜单选择外观时，确认模型、动作、表情配置同步切换。
