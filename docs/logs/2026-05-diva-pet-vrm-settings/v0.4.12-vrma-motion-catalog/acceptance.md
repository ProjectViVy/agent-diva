# Acceptance

## User Steps

1. Open the Diva pet settings panel.
2. Open 动画 and confirm the expanded motion list includes the new actions.
3. Preview one-shot actions such as 鼓掌, 再见, 生气, or 思考.
4. Open 外观 and create or edit a user appearance.
5. Confirm 环顾四周, 放松, and 困倦 are available in the standby motion set.
6. Save the appearance and switch to it to confirm the selected standby set is applied.
